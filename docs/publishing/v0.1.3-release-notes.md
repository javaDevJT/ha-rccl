# Royal Caribbean `v0.1.3`

## Changes

- Fixes Club Royale calendar-card bars collapsing to the departure day when a
  sailing row has a missing or stale return date.
- Derives Club Royale sailing duration from `N Night` itinerary labels when
  numeric duration fields are not available.
- Adds automatically managed calendar entities for each Club Royale offer code.
- Removes stale per-offer calendar entities when RCCL no longer returns that
  offer code.

## Notes

- The aggregate `Club Royale offer expirations` calendar remains available.
- Per-offer calendar entities make individual offer-code expiration dates easier
  to discover and select in Home Assistant's calendar UI.
