# Royal Caribbean `v0.1.2`

## Changes

- Adds automatically managed Club Royale offer-code date sensors.
- Adds a Club Royale offer-expiration calendar entity with one all-day event per
  offer expiration date.
- Publishes offer metadata on each offer-code sensor, including offer type,
  guest occupancy, sail-by date, ships, itineraries, cabin guarantees, departure
  ports, and linked sailing ids.
- Removes stale Club Royale offer-code sensors automatically when an offer is no
  longer returned by RCCL.

## Notes

- Club Royale sailing rows remain offer-scoped internally so duplicate RCCL
  voyages can still be filtered correctly by offer.
- The custom calendar card still de-duplicates visible duplicate sailings; this
  release adds Home Assistant entities and calendar events for the offer codes
  themselves.
