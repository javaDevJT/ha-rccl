# Royal Caribbean `v0.1.4`

## Changes

- Removes the per-offer Club Royale calendar entities that duplicated each
  offer-code date sensor in Home Assistant.
- Cleans up legacy per-offer calendar entities that were created by `v0.1.3`.
- Keeps the aggregate `Club Royale offer expirations` calendar for expiration
  events.
- Renames Club Royale offer date sensors to prefer the friendly offer name and
  include the offer code when available.

## Notes

- The per-offer date sensors remain the primary offer-code entities.
- The aggregate calendar still exposes offer expiration events without adding a
  second entity for every offer code.
