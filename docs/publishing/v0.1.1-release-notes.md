# v0.1.1 Release Notes

## Fixed

- Fixed Club Royale sailing rows being collapsed across offers that shared the
  same RCCL voyage id. Offer+sailing rows now use an offer-scoped id while
  preserving RCCL's original voyage id as `source_sailing_id`.
- Kept duplicate offer rows available for filtering while showing only one
  visible calendar bar per voyage. When filters leave multiple matching offers
  for the same voyage, the card displays the soonest-expiring offer.
